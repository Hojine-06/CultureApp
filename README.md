# CultureApp

benin_culture_app — Application Flutter

Ce dépôt contient l'application mobile "CultureApp" pour la promotion
du patrimoine culturel du Bénin.

Getting started
---------------

This project is a Flutter application scaffold. To run the app locally:

```powershell
cd C:\Users\Admin\desktop\projetmobile\CultureApp
flutter pub get
flutter run
```
